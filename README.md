# Dovahkiin

### Technology being used:

- Express JS [with ES6 (ES 2015) syntax]
- Gulp
- Bootstrap (SASS)
- Angular JS

### Quick Start

```sh
git clone https://github.com/krisanalfa/dovahkiin.git
cd dovahkiin
bower install
npm install
npm run dev
```

> **NOTE:** Use `npm run prod` on production server
